# 北极星新前端

这是北极星 Web 界面的正式重构基线。

## 技术栈

- React 19
- Next.js 16
- TailwindCSS v4
- TypeScript
- shadcn/ui

## 目录说明

- `src/app/`：页面与代理路由
- `src/components/polaris/`：北极星业务组件与工作台壳层
- `src/components/ui/`：shadcn/ui 组件
- `src/lib/`：类型、API 工具、代理辅助

## 本地运行

默认依赖现有 FastAPI 服务作为能力层，地址为：

- `http://127.0.0.1:8888`

如需修改，可设置环境变量：

- `POLARIS_API_BASE_URL`

### 开发命令

```bash
npm run dev
```

### 校验命令

```bash
npm run lint
npm run build
```

## 约束

- 新的 Web 页面优先开发在这里
- 不再继续把旧模板页作为新的视觉和交互基线
- 所有 UI 改动遵循黑白灰主色、浅蓝 CTA、圆角、留白、轻阴影、自然过渡
- 文案排版默认遵循“单行优先”原则：一句话在当前容器内能够完整表达时，不要为了样式强拆成两行或多行；纯说明文本默认不额外加边框，除非它承担状态或交互职责
- 浮动卡片、弹窗、抽屉这类临时容器如果因为宽度过小导致核心文案或步骤说明被迫碎成多行，优先放大容器或调整布局，不要默认压缩内容
- 仅确认类轻交互允许使用小弹层；流程说明、业务编排、详情查看、表格预览、表单录入默认使用中大尺寸弹层，不能直接沿用组件默认窄宽度
- 内容型弹层在桌面端优先给到 `lg` 级别以上宽度，并显式设置宽度上限；如果正文、步骤卡或表格明显拥挤，先扩容器，再调整排版
