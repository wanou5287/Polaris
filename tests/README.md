# 财务报表自动化生成系统 - 测试套件

## 📋 测试概述

本测试套件为财务报表自动化生成系统提供核心功能测试，专注于关键业务流程的验证。测试套件采用简化设计，只保留最核心的测试模块。

## 🧪 核心测试模块

### 1. 核心集成测试 (`test_core_integration.py`)
- **功能**: 测试系统的完整业务流程
- **测试内容**:
  - 用友API集成测试
  - 数据拉取服务测试
  - Excel导出服务测试
  - 报表生成服务测试（加密压缩）
  - 完整工作流程测试（端到端）
- **特点**: 端到端测试，验证整个业务流程，包含加密压缩功能

### 2. 用友接口集成测试 (`test_yonyou_integration.py`)
- **功能**: 测试用友API的完整集成流程
- **测试内容**:
  - access_token获取测试
  - 凭证查询测试
  - 数据解析和格式化测试
  - JSON数据保存测试
- **特点**: 真实API调用，包含详细日志输出

### 3. 钉钉通知和加密功能测试 (`test_notification_and_encryption.py`)
- **功能**: 测试钉钉通知和加密功能
- **测试内容**:
  - 钉钉Webhook连接测试
  - 文件下载通知测试
  - 加密ZIP文件功能测试
  - 密码验证测试
  - 完整报表生成流程测试
- **特点**: 综合测试，验证通知和加密功能

## 🚀 运行测试

### 运行端到端接口测试（GET 调用）
```bash
# 启动服务（另一个终端）
uvicorn main:app --host 0.0.0.0 --port 8888 --reload

# 运行端到端接口测试（GET 方式）
python tests/test_e2e_api.py
```

### 直接通过浏览器或curl调用
```bash
curl "http://127.0.0.1:8888/financial/generate-report?makeTimeStart=2025-09-01&makeTimeEnd=2025-09-30"
```

## 🔧 测试配置

### 环境要求
- Python 3.8+
- 项目依赖已安装
- 数据库连接正常
- 服务启动正常

### 测试数据
- 使用模拟数据进行测试
- 不依赖外部服务
- 测试数据自动清理
- 支持数据打印和日志输出

## ⚠️ 注意事项

1. **API测试**: 需要确保应用服务正在运行
2. **数据库测试**: 需要确保数据库连接正常
3. **外部服务**: 部分测试可能因为外部服务不可用而跳过
4. **测试隔离**: 每个测试模块独立运行，互不影响
5. **数据拉取测试**: 包含详细的数据拉取日志输出

## 🐛 故障排除

### 常见问题
1. **导入错误**: 确保项目根目录在Python路径中
2. **数据库连接失败**: 检查数据库配置和连接
3. **服务启动失败**: 检查端口占用和依赖安装
4. **测试超时**: 检查网络连接和外部服务状态
5. **数据拉取失败**: 检查用友接口配置和网络连接

### 调试建议
1. 查看详细错误信息
2. 检查配置文件
3. 验证依赖安装
4. 确认服务状态
5. 使用数据拉取集成测试查看详细日志
