启动与联调

DataAgent 目前还保持独立运行，不强行并进北极星主进程。这样接入风险更小，也更方便我们后续决定是代理转发，还是继续作为独立服务。

Streamlit 与 API 默认都走本机端口。如果你希望外网同事也能直接访问，下一步需要给 DataAgent 也做统一代理或反向转发。
